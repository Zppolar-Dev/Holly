Política de Privacidade

Última atualização:

1. Informações Coletadas

A Holly ("o Bot") pode coletar e armazenar as seguintes informações quando você o utiliza:

  • ID do usuário do Discord
  • ID do servidor do Discord
  • Nomes de usuário e discriminadores
  • Configurações específicas do servidor
  • Dados de uso (como comandos executados)

2. Uso das Informações

As informações coletadas são usadas para:

  • Fornecer e melhorar os serviços do Bot
  • Manter a funcionalidade básica do Bot
  • Personalizar a experiência do usuário
  • Garantir a segurança e prevenir abusos

3. Compartilhamento de Dados

Não vendemos, alugamos ou compartilhamos seus dados pessoais com terceiros, exceto quando:

  • Exigido por lei
  • Necessário para proteger nossos direitos
  • Com seu consentimento explícito

4. Segurança dos Dados

Implementamos medidas de segurança para proteger suas informações, mas nenhum sistema é 100% seguro. Não podemos garantir a segurança absoluta dos dados.

5. Seus Direitos

Você tem o direito de:

  • Solicitar acesso aos seus dados armazenados
  • Solicitar a correção de dados incorretos
  • Solicitar a exclusão de seus dados pessoais

Para exercer esses direitos, entre em contato através do nosso servidor de suporte.

6. Alterações nesta Política

Podemos atualizar esta Política periodicamente. A versão mais recente estará sempre disponível nesta página.

7. Contato

Para quaisquer dúvidas sobre privacidade, entre em contato através do nosso servidor de suporte no Discord.
